Mode File:
r: Baca doang (kalo file gaada, error).
w: Tulis baru (isi lama ilang).
a: Tambah data di bawah (isi lama aman).
with open(): Otomatis tutup file. Wajib pake biar data ga korup.
\n: Enter / baris baru.
CSV:
reader: Bongkar isi CSV.
writer: Masukin data ke CSV.
next(reader): Skip baris judul (paling atas).
writerow: Masukin 1 baris.
writerows: Masukin banyak baris sekaligus.
Angka:
int(): Angka bulet (1, 2, 3).
float(): Angka koma (3.14, 10.5). perbandinganya keduanya kalau rata2 atau kemungkinan hasilnya bisa desimal pakek selalu float kecuali hasilnya pasti bilangan bulat baru int
